The Iki Toolkit extends functionality of Iki Themes, providing custom post types and more.

== Description ==

Iki Toolkit provides support for portfolio, team member and wonder grid custom post types.

== Installation ==
This plugin can be installed directly from your site.

1. Log in and navigate to Plugins & Add New.
2. Type "Iki Toolkit" into the Search input and click the "Search" button.
3. Locate the in the list of search results and click "Install Now".
4. Click the "Activate Plugin" link at the bottom of the install screen.

It can also be installed manually.

1. Download the "Iki Toolkit" plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the "Plugins" screen.
4. Locate "Iki Toolkit" in the list and click the "Activate" link.



==Frequently Asked Questions==

= What is this plugin and why do I need it? =

This plugin provides extra functionality for Iki Themes. It provides additionall custom post types that can be utilized by the theme.
